# Dexoshi

## Tweet Commands

```
# Get info about @user
@dexoshi info <@TWITTER_HANDLE>

# Gift to @user token ID #1
@dexoshi gift <@TWITTER_HANDLE> 1

# Burn token ID 1
@dexoshi burn <TOKEN_ID>

# Transfer to 0x address
@dexoshi trasnfer <ADDRESS> <TOKEN_ID> <AMOUNT>
```
