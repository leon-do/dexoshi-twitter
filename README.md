```
🤓 Get info
@dexoshi info <@TWITTER_HANDLE>

🎁 Gift token
@dexoshi gift <@TWITTER_HANDLE> <TOKEN_ID>

🫶 Merge token 2 + 3 = 6
@dexoshi burn <TOKEN_ID_1> <TOKEN_ID_2>

📨 Transfer to address
@dexoshi transfer <0xADDRESS> <TOKEN_ID> <AMOUNT>

🔥 Burn token
@dexoshi burn <TOKEN_ID>
```
